# Stacker

A faithful recreation of the classic arcade Stacker game built with Vite, TypeScript, and HTML5 Canvas. Optimized for mobile.

## Play

[Play online](https://stacker-tawny.vercel.app)

## Features

- Classic arcade gameplay - stack blocks to reach the top
- Increasing speed as you climb higher
- Minor prize at row 10, major prize at row 15
- Top rows force single-block precision (just like the real arcade)
- Touch, mouse, and keyboard support
- Haptic feedback on mobile devices
- Sound effects using Web Audio API
- High score persistence
- Celebration animation on winning
- PWA support - install to home screen

## Controls

- **Mobile**: Tap anywhere
- **Desktop**: Click or press Space/Enter

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- Vite
- TypeScript
- HTML5 Canvas
- Web Audio API
